## Technologies used

- [Express](https://expressjs.com/)

## Project setup

1. Run `npm run setup`
1. Run `npm run build`
1. Add `APP_PORT` to your environment variables
1. Run `npm run run`

The server is running on `localhost:APP_PORT`.
